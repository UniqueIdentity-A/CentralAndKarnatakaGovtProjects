## 🧠 What This Skill Does

This skill transforms Claude into a **Government Scheme Intelligence Dashboard** — not just a search tool. It actively discovers, analyzes, and visualizes government welfare schemes from official sources.

### Capabilities

| Feature | Description |
|---|---|
| 📋 **Scheme Discovery** | Lists all Central & Karnataka government schemes from official portals |
| 💎 **Hidden Gems** | Finds lesser-known schemes with low public awareness but high value |
| ⚠️ **Deadline Alerts** | Identifies schemes closing soon with application deadlines |
| 🆕 **New Launches** | Tracks recently launched schemes from PIB and ministry portals |
| 📊 **Analytics Dashboard** | Budget allocations, beneficiary stats, category breakdowns |
| 🔍 **Smart Filtering** | Filter by category: farmers, students, women, SC/ST, startups, and more |

## 🏛️ Official Data Sources

This skill fetches live data only from trusted official portals:

**Central Government**
- [india.gov.in](https://www.india.gov.in) — National scheme directory
- [pib.gov.in](https://pib.gov.in) — Press releases for new launches
- [mygov.in](https://www.mygov.in) — Citizen engagement portal
- [data.gov.in](https://data.gov.in) — Open government datasets

**Karnataka Government**
- [karnataka.gov.in](https://www.karnataka.gov.in) — State scheme listing
- [sevasindhu.karnataka.gov.in](https://sevasindhu.karnataka.gov.in) — Application portal
- [cmodashboard.karnataka.gov.in](https://cmodashboard.karnataka.gov.in) — Beneficiary stats
